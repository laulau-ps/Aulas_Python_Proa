O Jinja2 pega seu HTML, injeta dados e lógica usando os delimitadores, e entrega uma página web final e dinâmica.
Para dar as instruções ao Jinja2 dentro do HTML, você usa três tipos de sinais especiais (todos estão dentro de chaves, não coloquei aqui pra não dar gatinho):
% ... % Usado para lógica e instruções. Ex: Criar loops (for) ou tomar decisões (if). Não aparece na saída.
{ ... } Usado para imprimir um valor. Exibe o conteúdo de uma variável.
# ... # Usado para comentários internos. O motor ignora totalmente essa parte.
